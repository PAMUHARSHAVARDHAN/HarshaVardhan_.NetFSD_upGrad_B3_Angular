FAQ

1.what is Html

HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the World Wide Web

2.what is css

CSS (Cascading Style Sheets): The skin/styling of a website. It dictates colors, fonts, layout, and responsiveness (how it looks).

3.what is javascript

JavaScript (JS): The brain/functionality of a website. It makes pages interactive, handling things like dropdown menus, animations, and real-time updates.

4.what is jquery

jQuery: A fast, small JavaScript library designed to simplify HTML DOM traversing, event handling, and AJAX interactions. It makes complex JS tasks easier to write.

5.what is bootstrap

Bootstrap: A front-end framework (using CSS and JS) that provides pre-styled components (buttons, navbars, grids) for fast, responsive, mobile-first web development
